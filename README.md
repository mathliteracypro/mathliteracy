## 🛠️ Tech Stack (Documented)

- **Runtime**: Bun
- **Backend**: Express + Vite
- **Frontend**: React 19 + TypeScript
- **Database**: Firebase Firestore
- **Authentication**: Firebase Auth
- **AI**: Google Gemini API
- **Styling**: Tailwind CSS 4
- **Hosting**: InfinityFree
